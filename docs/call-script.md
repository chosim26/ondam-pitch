# 온담 전화서비스 통화 가이드

> 대본이 아닌 흐름 가이드. 자연스러운 대화가 핵심이다.
> 어르신마다 성격·상황이 다르니 유연하게 조절할 것.

---

## 1. 첫 통화 (소개 + 라포 형성)

### 목표
- 이 전화가 뭔지 이해시키기
- 경계심 풀기
- 다음 통화 약속 잡기

### 흐름

**① 인사 + 신원 밝히기**
> "안녕하세요 어머님/아버님, 저는 '온담'이라는 안부전화 서비스의 ○○○입니다."

**② 자녀가 신청했다는 것 전달**
> "따님/아드님 ○○○님이 어머님 안부가 궁금하시다고 저희한테 부탁을 하셨어요."
> "○○○님이 바쁘셔서 자주 전화를 못 드리는 게 마음에 걸리셨나 봐요."

- 핵심: **자녀의 마음**이 담긴 전화라는 걸 전달
- "회사에서 전화했다"보다 "자녀분이 부탁하셨다"가 훨씬 자연스러움

**③ 서비스 간단 설명**
> "저희가 매주 한 번씩 어머님께 안부 전화를 드리고,
> 어머님 근황을 따님/아드님께 간단히 전달해드리는 서비스예요."
> "편하게 수다 떠는 거라고 생각하시면 돼요!"

- 지나치게 설명하지 않기 — "전화 올 거다" 정도만 이해하면 OK
- "리포트", "서비스" 같은 딱딱한 단어 피하기

**④ 가벼운 대화로 전환**
> "요즘 날씨가 많이 풀렸는데, 산책은 좀 하고 계세요?"
> "오늘 뭐 드셨어요? 저는 아침에 ○○ 먹었는데요~"

- 첫 통화는 5~10분이면 충분
- 억지로 길게 끌지 않기 — 자연스러운 마무리가 더 좋은 인상

**⑤ 다음 통화 예고 + 마무리**
> "어머님, 저 다음 주 ○요일에 또 전화 드려도 될까요?"
> "혹시 전화 받기 편한 시간대가 있으세요?"
> "그럼 ○요일 ○시쯤 다시 전화 드릴게요. 오늘 통화 즐거웠습니다!"

## 2. 정기 통화 (주 1회)

### 목표
- 어르신의 일주일 근황 파악
- 기분·건강·활동·하고 싶은 말 체크
- 편한 대화 상대가 되어주기

### 자연스럽게 확인할 4가지

| 영역 | 질문 예시 | 체크 포인트 |
|------|----------|------------|
| **기분** | "이번 주는 기분이 어떠셨어요?" | 우울감, 외로움, 짜증, 기쁜 일 |
| **건강** | "몸은 좀 어떠세요? 잘 주무시고?" | 수면, 식사, 통증, 병원 방문 |
| **활동** | "이번 주에 어디 다녀오셨어요?" | 외출, 운동, 취미, 사회활동 |
| **하고 싶은 말** | "자녀분께 전해드릴 말씀 있으세요?" | 자녀에게 전할 메시지, 요청사항 |

### 대화 팁

- **경청이 핵심.** 어르신이 말씀하실 때 끊지 않기
- **맞장구:** "아~ 그러셨구나", "그거 좋으시겠다", "힘드셨겠어요"
- **반복 화제 OK:** 매번 같은 얘기 하셔도 처음 듣는 것처럼 반응
- **조언 자제:** 물어보시기 전엔 조언하지 않기. 들어주는 것만으로 충분
- **시간:** 보통 10~20분. 짧아도 괜찮고, 대화가 잘 되면 길어져도 OK
- **개인정보:** 어르신의 개인적인 이야기는 리포트에 필요한 것만 기록

### 주의사항

- 건강 문제 언급 시 → "병원은 가보셨어요?" 정도만 (진단/처방 절대 X)
- 우울감이 심해 보일 때 → 자녀에게 별도 연락 (리포트에도 표기)
- 전화를 안 받으실 때 → 다른 시간에 1회 재시도, 그래도 안 되면 자녀에게 알림
- 서비스 중단 원하실 때 → "네, 알겠습니다" 존중 후 자녀에게 전달

## 3. 통화 후 리포트 작성

### 자녀에게 카톡으로 전달하는 주간 리포트 양식

```
[온담 주간 안부 리포트]
📅 통화일: 2026년 O월 O일 (O요일)
📞 통화 시간: 약 OO분

😊 기분: (한 줄 요약)
🏥 건강: (한 줄 요약)
🚶 활동: (한 줄 요약)
💬 하고 싶은 말: (어르신이 전해달라고 한 말)

📝 상담사 메모: (특이사항이나 자녀가 알면 좋을 것)
```

### 작성 원칙

- **긍정적 톤 유지** — "하루종일 누워계셨대요" (X) → "이번 주는 좀 쉬셨대요. 몸이 좀 피곤하신 것 같아요" (O)
- **사실 기반** — 추측이나 진단 넣지 않기
- **짧고 읽기 쉽게** — 자녀도 바쁜 사람. 30초 안에 읽을 수 있는 분량
- **하고 싶은 말은 그대로** — 어르신 말투 살려서 전달하면 감동 포인트

## 4. 상황별 대응

### 어르신이 경계하실 때
> "사기 전화 아니에요~ 따님 ○○○님한테 확인해보셔도 돼요."
> "제 전화번호 010-8326-8528 저장해두시면 다음에 안심하고 받으실 수 있어요."

### 어르신이 우실 때
- 말없이 기다려드리기 (침묵 OK)
- "많이 서운하셨구나..." 공감
- 억지로 밝게 전환하지 않기

### 통화를 안 끊으려 하실 때
> "어머님 저도 더 얘기하고 싶은데, 다음 주에 또 전화 드릴게요!"
> "다음에 이어서 들려주세요~"

### 자녀 험담을 하실 때
- 동조하지도, 반박하지도 않기
- "그런 마음이 드셨구나요" 정도로 수용
- 리포트에는 중립적으로 기록 ("자녀와 소통을 더 원하시는 것 같습니다")

### 선물/금전 요청하실 때
> "저희는 안부 전화 서비스라서 그 부분은 어려운 점 양해 부탁드려요."
> 필요하면 자녀에게 전달
