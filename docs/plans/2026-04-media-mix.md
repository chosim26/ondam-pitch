## 의사결정 트리

```
3월 말 데이터 리뷰
├── 모임 소재 CTR > 1% → W3에서 예산 증액 (₩25K→₩35K)
├── 모임 소재 CTR 0.5~1% → 소재 변형 테스트 후 판단
├── 모임 소재 CTR < 0.5% → 소재 전면 교체
│
├── 고민전화 CTR > 모임 CTR → 별도 캠페인 분리, 전용 랜딩 강화
├── 고민전화 CTR ≈ 모임 CTR → 소재 변형 추가 테스트
├── 고민전화 CTR < 모임 CTR → 카피 변경 or 중단
│
├── 릴스 ThruPlay > 15% → 릴스 예산 비중 확대
└── 릴스 ThruPlay < 10% → 릴스 소재 교체 or 피드 집중
```
